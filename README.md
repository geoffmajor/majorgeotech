# Major Geotech PLLC Website

This repository contains the public website for **Major Geotech PLLC**, a geotechnical engineering consultancy providing independent technical review, scoping studies, and forensic services for surface mining and pipeline projects.

The site is a single-page static website hosted on **GitHub Pages** and served at:

https://majorgeotech.com

---

## Overview

Major Geotech PLLC provides specialized geotechnical engineering expertise, with a focus on:

- Independent technical reviews
- Open pit scoping and evaluation studies
- Forensic geotechnical investigations
- Support for surface mining and pipeline projects

This repository contains only the website source code and related assets.

---

## Hosting

The site is hosted using **GitHub Pages** with a custom domain.

- Repository: project site (not a user site)
- Branch: `main`
- Custom domain: `majorgeotech.com`
- Static assets only (HTML, CSS, JavaScript)

Legacy URLs from the previous multi-page site are handled via redirect pages to preserve continuity and search engine indexing.
